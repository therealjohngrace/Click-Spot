# Click-Spot
A tool for seeing where you clicked at. Only works on PC for now.
Note: This tool is in Alpha. It may not show where you clicked. The customizing system may be bugged, but we will work on making it fixed and add more pre-made click markers. See sections down below.


# Markers
Markers can sometimes not work, which we will work on fixing that. A marker is customizable. You can
-Tweak settings to increase the radius
- Make it so that when you click it makes a sound which will not be a mouse click one
- Make your cursor pulse when clicking


# Customization
Customizing is an essential part of making this tool fit just for you! If you don't like the plain circle marker, then you are free! I put thought into making it so that the user has free will to customize their click marker.
## Colors
You prolly don't like plain and simple red. Unless you do. So we made the color customizable! Now you can have a one colored, 3 colored, 5 colored, or rainbow colored marker! You can also type a hex code to get a color too.
### Swatches
Want to change markers every day but don't wanna keep trying to adjust the color perfectly! No problem! Swatches are here to save your day! Its simple! Just make a new swatch, click the slot you want to put the color in, and then you've got a perfectly saved color paletee for you!
## Shape
Shape is kind of non essential, but if you hate circles, its your field day! You can change the shape to a square, rectangle, triangle, and more!
## Image
Want to have the free-est will you've ever had? Well strap on in! You can make an image your click marker instead! Your a FNAF fan and want a freddy marker? No problem! Like Nyan Cat? NO PROBLEM!
